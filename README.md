# Intelligent Intrusion Detection System using Machine Learning
## 📌 Project Overview

- The Intelligent Intrusion Detection System (IDS) is a Machine Learning based web application developed to detect and classify suspicious network traffic activities.

This system helps identify whether incoming network traffic is:

- Normal
- Intrusion / Attack

The project provides:

- Real-time prediction
- Feature importance visualization
- Confusion matrix analysis
- Interactive dashboard for monitoring cyber threats

# ⚙️ Installation & Setup
## 1️⃣ Clone the Repository
- git clone https://github.com/your-username/your-repository-name.git [COMMAND]

## ▶️ Run Frontend
### Open frontend folder

- cd frontend [COMMAND]

### Install dependencies

- npm install [COMMAND]

### Start React application

- npm start [COMMAND]

### Frontend runs on:

- http://localhost:3000


# ▶️ Run Backend
## Open backend folder

- cd backend [COMMAND]


## Create virtual environment

- python -m venv venv [COMMAND]


## Activate environment

### Windows

- venv\Scripts\activate [COMMAND]

### Linux / Mac

- source venv/bin/activate [COMMAND]

## Install dependencies

- pip install -r requirements.txt [COMMAND]

## Run FastAPI server

- uvicorn main:app --reload [COMMAND]

### Backend runs on:

- http://127.0.0.1:8000
